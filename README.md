# Banuba (banuba)
Banuba is an AR and AI technology company providing the Face AR SDK for augmented reality face effects, beauty filters, and virtual try-on experiences. The SDK supports iOS, Android, Web (HTML5), Windows, macOS, Unity, Flutter, and React Native.

## APIs

### Banuba Face AR SDK
The Banuba Face AR SDK provides AR face effects, beauty filters, and face tracking for mobile, web, and desktop applications. Includes real-time face detection, 3D face tracking, background segmentation, and customizable AR effects. Integration is via native SDK rather than REST API.

## Features

| Name | Description |
|------|-------------|
| Face AR Effects | Real-time AR face masks, filters, and accessories for live and recorded video. |
| Beauty Filters | Skin smoothing, face reshaping, eye enlargement, and makeup filters. |
| 3D Face Tracking | Precise 3D face landmark tracking for accurate AR effect placement. |
| Background Segmentation | Real-time background removal and replacement for video calls. |
| Virtual Try-On | Virtual makeup, glasses, hair color, and accessory try-on experiences. |
| AI Face Detection | ML-powered face detection with age, gender, and emotion analysis. |
| Cross-Platform SDK | Native SDKs for iOS, Android, Web, Windows, macOS, Unity, Flutter, and React Native. |
| Custom Effects Builder | Banuba Effect Player for creating custom AR effects without coding. |
